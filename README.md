# hi.cami — Portafolio Audiovisual

Portafolio de **Camila Quiceno**, Filmmaker & Editor especializada en comerciales, documentales y cortometrajes.

### Stack

- **React 19** + **Vite 8**
- **Tailwind CSS 4** — diseño monocromo premium
- **Framer Motion** — animaciones, parallax, transiciones
- **Oxlint** — linting

### Desarrollo

```bash
npm run dev      # servidor local
npm run build    # build para producción
npm run preview  # previsualizar build
```

### Deploy

Conectado a Vercel. Cada push a `main` hace deploy automático.

### Dominio

**hi.cami** — apuntado desde Vercel Dashboard > Settings > Domains.
